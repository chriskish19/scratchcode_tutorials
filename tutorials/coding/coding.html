Step 2: Compile a simple program

We will start with a goal of making a program that outputs "Hello World" to the console. Here is the code in c++:

#include <iostream> 
using namespace std;
        
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

This is called an include directive it tells the C++ pre-processor to place the entire contents of the file "iostream.h" on the same line. In simple terms it's a copy and paste command. The pre-processor is part of the C++ language and is a program that is run first before the code is compiled. Compiling code is a process by which a program called a compiler parses the code(symbols) into an executable program which then can be run in an OS(operating system) enviroment. The file extension ".exe" is used in Microsoft Windows. The packages you installed for C++ in Step 1 contained a compiler and pre-processor program.

#include <iostream>

The "using" directive in C++ is a powerful feature that affects how the compiler resolves names in your code. It can be used for resolving namespaces. Namespaces are a way to add an extra name to parts of a program and serves a purpose of avoiding name clashes. The standard library uses "std" as their namespace. By declaring "using namespace std" we are telling the compiler to resolve all instances of the standard library. In C++ all statements must end with a semi-colon similar to how sentences end with a period.

using namespace std;

Without the "using namespace std". The "std::" is must be prefixed to the "cout" and "endl" statements, in order to use them. "::" is called the scope resolution operator. In C++ some character symbols are operators. Like how "+" and "-" in math are symbols that indicate what to do with two numbers on either side. In C++ the "::" operator performs a check in the above scope to find a class, enum or namespace name. We will learn about classes and enums later in the tutorial.

if else statements

These statements control program flow. If something is true then do this else do this. It is also called branching. Compile and run this program, then change on to false, recompile and run. Notice it outputs "switch is off" when on is false and when on is true it outputs "switch is on".

// ifelse C++ example program
#include <iostream>
int main(){
    // this is a comment, it is ignored by the compiler and is not executed
    /* this is also a comment */
    bool on{true}; // change to: bool on{false};
    if(on){
        // if on is true this will execute any code inside the curly braces
        std::cout << "switch is on" << std::endl; // in c++ a semi-colon ends a line of code, like a period in a sentence.
    }
    else{
        // if on is false this will execute any code inside the curly braces
        std::cout << "switch is off" << std::endl;
    }
    // std::cout << "text to be displayed in the terminal" << std::endl;
    // std::cout is a function that can accept almost any type for which the << operator is overloaded, the type we passed in is a const char*
    // we will learn more about this later 
    
    return 0; // main function must return an integer value, to indicate we sucessfully executed to the end of the main function we return the value 0
}
