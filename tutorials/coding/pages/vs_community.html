Get it here. When you get to the install option page select check all the boxes in red.

Once you open visual studio community 2022 select Create new project

In the seach bar type c++ empty project and select Empty Project: start from scratch with C++ for Windows. Provides no starting files.

Name the project HelloWorld and click create.

Right click on Source Files (Top right) then Add->New Item or press CTRL+SHIFT+A. Select C++ File name it main then click add.

Copy the Hello World program from Coding basics. Click the Local Windows Debugger button.

A console window should appear with "Hello World!" printed
